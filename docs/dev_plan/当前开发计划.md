## 验收标准（DoD）

1. create session 具备 user/ip 限流与幂等回放能力。
2. create session 写入审计，且 replay 可追溯。
3. `topicId` 超界输入返回稳定业务错误，不出现符号翻转语义。
4. 主题删除竞态不再产生“语义错位”的写入路径。
5. OpenAPI 与运行态关键错误谱系一致。
6. 最小回归测试通过，且无编译回归。

## 风险与回退

1. 风险：新增幂等表与审计表可能引入迁移差异。
   - 回退：migration 独立、可回滚；handler 保持 fail-open 锁策略避免业务阻断。
2. 风险：时间语义收紧可能影响“补历史场次”脚本。
   - 回退：如业务确需补录，后续单独提供受控内部入口。
